# Rent 2 Own Cribs (R2OC)

**Summary**: Marketing brand for rent-to-own properties and content. Social media automated via Content AI. Facebook presence: R2OC Page + "Lubbock Rent-A-Home Homes Available" group.

**Sources**: CLAUDE.md seed; transcript-2026-06-12-morning-meeting-josh-mostafa.md

**Last updated**: 2026-06-22

---

External-facing brand for [[wih]]'s rent-to-own properties and educational content. Content pipeline is being automated via [[content-ai]] using Google Sheets + daily automation.

## Facebook presence
- **Facebook Page**: Rent 2 Own Cribs (Rent-A-Home Cribs) — the brand page.
- **Facebook Group**: "Lubbock Rent-A-Home Homes Available" — the community/listing group.
- **Posting SOP** (confirmed Jun 12, 2026): Post as the R2OC Facebook PAGE to the Facebook GROUP using cross-posting (not sharing). Cross-posting creates an independent post vs. sharing, which drastically underperforms in reach.
- **Marketplace listing rule**: Always change the listing title on Facebook Marketplace. A price listing (e.g., $9,200) with no descriptive title looks like a scam — one listing triggered a scam report. Title must include context (e.g., "Rent to Own – $9,200 Down").
- **Highest-traffic drivers**: text blasts + bandit signs spike traffic more than Facebook alone.
- **Content format for group**: video (not still images) + long description. No photos if posting full video.
