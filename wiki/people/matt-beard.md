# Matt Beard

**Summary**: AI-augmented wholesale framework. Primary external reference for WIH's AI automation strategy.

**Sources**: CLAUDE.md seed

**Last updated**: 2026-06-03

---

The primary external model Joshua is studying for [[ai-automation-strategy]]. Matt Beard's framework for AI-augmented wholesale operations is the blueprint for how [[ghl]] becomes the hub for automated deal sourcing.

His system is branded the **"No Fluff" model**, run through a community of **2,000+ members**. The detailed framework lives on [[no-fluff-model]].

## The 3 phases
1. **Lead Generation** — direct-to-agent off-market outreach (see [[agent-outreach]]), with leads bucketed into **Tier 1 / Tier 2 / Tier 3** plus a **"Cash Agent Dead"** bucket for sorting/prioritization.
2. **Lead Conversion — the 3 Cs:** **Call → Comp → Close.**
3. **Disposition** — buyer-side movement measured by the **time-to-buyer (TTB)** KPI (see [[disposition]]).

**Economics:** roughly **$100–150/deal** to run the machine.
