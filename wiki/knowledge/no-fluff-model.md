# No Fluff Model

**Summary**: [[matt-beard]]'s "No Fluff" wholesale system — the canonical end-to-end framework Joshua is building [[vince-ai]] + [[wih-app]] + [[ghl]] around, spanning lead generation, conversion, and disposition.

The "No Fluff" model is [[matt-beard]]'s complete AI-augmented wholesale system. It is the spine of Joshua's wholesale automation build — the framework [[vince-ai]], [[wih-app]], and [[ghl]] are all being designed against.

## The three phases

### Phase 1 — Lead Generation
Text and email agents run at scale. An AI bot filters every inbound reply into buckets:
- **No answer** → follow-up sequence
- **Hostile** → "Cash Agent Dead" (remove/suppress)
- **Tier 1** (gave an address) → advance to Phase 2
- **Tier 2** (address didn't work out) → follow-up
- **Tier 3** (no deal but open) → a different follow-up track

Metrics: ~500 texts/day → 3–8 leads → 22–40% are off-market → 12–20 leads per deal. "Gold" = a property that is off-market **and** ugly **and** asking under Zestimate. The acquisition script that powers this phase lives in [[agent-outreach]].

### Phase 2 — Lead Conversion (the 3 Cs)
- **Call** — Lead Management Call within ≤6 hr. "The sale happens on the first call." Build rapport, get the lay of the land.
- **Comp** — underwrite within ≤12 hr.
- **Close** — only present to someone ready to decide; give a verbal ballpark first.

### Phase 3 — Disposition
See [[disposition]]. JV when you're new. Key KPI: **TTB (Time To Blast) < 12 hr = +400% sell likelihood**. Only 60–65% of signed contracts actually close.

## Economics
$100–150 cost per deal vs. an industry norm of $2–10K. 30–45 day ramp to first deal.
