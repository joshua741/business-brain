# Disposition

Disposition ("dispo") is the sell side of wholesale — Phase 3 of the [[no-fluff-model]]. It is increasingly run on autopilot by AI.

## Buyer tiers
- **A** = verified closer
- **B** = vetted + proof of funds
- **C** = interested but unvetted (everyone enters at C; AI texts/vets → human call → promote to B)

## Three core functions on autopilot
1. General buyer prospecting
2. Deal-specific prospecting
3. JV / co-wholesaler fallback

## The 5-day dispo process
- **Day 0** — prep
- **Day 1** — blast + build
- **Day 2** — outreach
- **Day 3** — walkthrough RSVP reminders (skipping this = 50% drop-off)
- **Day 4** — one-window walkthrough to create competition; $5K non-refundable EMD
- **Day 5** — decision day: Magic Question ("is that really the best you can do?"), select the top buyer at 12pm, send a FOMO backup-offer text

## KPIs
TTB (Time To Blast) < 12 hr; ≥70% of dispo deals should close. AI buyer texter example: "saw you did that flip off [street], still buying?"
